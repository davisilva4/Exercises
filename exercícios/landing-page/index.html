Chatubot

Deixe a sua comunidade segura de um jeito estiloso

O que ele faz por você?

1. Cria um sistema de verificação de Captcha para averiguar se os crias não são robos.

2. Possui um sistema de banimento automático para links suspeitos enviados ao bonde.

3. Sistema de avisos e banimento manual para os parças.
